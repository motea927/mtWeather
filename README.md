## :page_facing_up: 技術


- **Node.js**：利用V8引擎的程式語言，使用javascript語法
- **Express**：架設伺服及路由的簡易框架
- **axios**：基於Promise的HTTP庫，可以用在瀏覽器和node.js中
- **@line/bot-sdk**：LINE bot的官方sdk
